## ✨ About the Project

**Pragyantra** *(Sanskrit: "Instrument of Knowledge")* is a final year engineering project — a full-stack, AI-driven platform that personalizes the engineering education journey.

> Instead of generic content, every student gets a unique experience tailored to their **interests**, **academic background**, and **career goals** — all powered by **Google Gemini 1.5 Flash**.

The platform guides students through:
1. 🎯 Personalized interest assessment (AI-generated scenario questions)
2. 🤖 AI analysis of strengths, weaknesses & career fit
3. 🗺️ Step-by-step career roadmaps generated by AI
4. 📊 Branch-personalized dashboard with progress tracking
5. 💬 Real-time AI mentor chatbot for doubt resolution

## 🚀 Key Features

| Feature | Description |
|---|---|
| 🔐 **Auth System** | JWT-based secure login, registration & profile setup |
| 🎯 **Interest Assessment** | 10 AI-generated scenario questions (no right/wrong — purely preference-based) |
| 🧠 **AI Analysis Engine** | Gemini analyzes responses → returns strengths, weaknesses, skill scores, branch match % |
| 🗺️ **Career Roadmap** | Multi-path, step-by-step learning roadmap generated by Gemini per branch |
| 📊 **Personalized Dashboard** | Branch-specific skills, roadmap, activity feed, streak calendar |
| 📈 **Progress Page** | Radar + bar charts, roadmap timeline, skill gap analysis |
| 💬 **AI Mentor Chatbot** | Floating chat widget powered by Gemini — answers any engineering question |
| 📄 **Resume Builder** | Editable, downloadable resume auto-filled from profile data |
| 🏅 **Leaderboard** | XP-based student rankings |
| 👥 **Peer Match** | Find study partners with complementary skills |
| 🔥 **Streak Tracker** | Calendar-based daily login streak system |

## 🤖 AI Engine — Gemini 1.5 Flash

All AI features are powered by **Google Gemini 1.5 Flash** via the `@google/generative-ai` SDK.

### `generateInterestQuestions(interests, name)`
Generates 10 unique, scenario-based questions tailored to the student's stated interests.
- **Input**: `['coding', 'robotics', 'ai_ml']`, `"Sujal"`
- **Output**: Array of 10 questions, each with 4 options tagged to interest domains
- **Mode**: JSON mode (structured)

### `analyzeStudentPerformance(profile, interestResponses)`
The core intelligence of the platform — analyzes all 10 responses to predict career fit.
- **Input**: User profile + array of `{question, selectedText, selectedTag}`
- **Output**: `{ strengths[], weaknesses[], skillScores[], dominantInterest, explanation, branchRecommendations[{branch, matchPct, reason, color}] }`
- **Logic**: Tags are tallied → top tags map to branches → Gemini explains in natural language

### `generateCareerFlowchart(branch)`
Generates a multi-path, step-by-step learning roadmap for any engineering branch.
- **Input**: `"Electronics & Communication"`
- **Output**: `{ branch, paths[{title, color, steps[{label, desc, duration, skills[]}]}] }`

### `chatWithMentor(message, history)`
Powers the floating AI mentor chatbot.
- **Mode**: Conversational (no JSON mode)
- **System prompt**: "You are OneStop Mentor — helpful, concise, practical"

### `predictCareers(profile, analytics)`
Suggests top career roles with salary ranges and skill gaps.
- **Output**: `{ matches[{career, match_percent, gap_skills[], salary}], confidence }`

```json
{ "name": "Sujal Bhosale", "email": "sujal@mail.com", "password": "secret123" }
```

### `POST /api/auth/login`
```json
{ "email": "sujal@mail.com", "password": "secret123" }
// Returns: { token: "jwt...", user: {...} }
```

### `PUT /api/auth/setup-profile`
```json
{
  "interests": ["coding", "ai_ml", "robotics"],
  "marks10": 92, "marks12": 88,
  "cetPercentile": 98.5, "jeeRank": 12400,
  "branch": "Computer Science"
```

### `GET /api/assessment/interest-questions`
```
Authorization: Bearer <jwt>
```


## 📊 Database Models

### User
| Field | Type | Description |
|---|---|---|
| `name`, `email` | String | Auth credentials |
| `passwordHash` | String | bcrypt hashed |
| `profile.interests` | `[String]` | e.g. `['coding', 'ai_ml']` |
| `profile.marks10/12` | Number | Academic scores |
| `profile.branch` | String | Chosen engineering branch |
| `xp`, `streak`, `level` | Number/String | Gamification data |

### Assessment
| Field | Type | Description |
|---|---|---|
| `userId` | ObjectId | Reference to User |
| `answers[]` | Array | `{questionId, selectedOption, timeTaken}` |
| `analysisResult` | Object | Full Gemini analysis output |
| `mlRecommendations[]` | `[String]` | Top predicted branches |

```env
RATE_LIMIT_WINDOW_MS=900000
```

## ▶️ Running the Project

### Prerequisites
- Node.js ≥ 18
- MongoDB running locally on port `27017`
